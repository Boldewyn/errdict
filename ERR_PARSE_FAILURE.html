ERR_PARSE_FAILURE. int—(Despairing) an error code usually received during e-mail conversations, when a dialog partner utters incoherent data that has virtually nothing to do with the topic of the previous e-mail or the overall conversation.

If this error code appears regularly, might turn the dialog partner into a green muscular raging monster. See also reasoning and Oh, look! A squirrel!.
